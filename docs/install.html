Install (30 seconds)

  1. Open the downloaded .dmg and drag Music Digest onto the Applications folder.
  2. Double-click Music Digest to launch it. It's signed with an Apple Developer ID and notarized by Apple, so it opens like any other app — no security prompts, no extra steps.
  3. That's it — the app opens. Launch it any time from Applications or Spotlight.

First launch

  1. Enter your name & emailYour name greets you in the digest; your email is where it gets delivered. No API keys, no payment — it's all handled for you.
  2. Click “Connect Spotify”Links your account so the app can build your daily playlist automatically.
  3. Hit ▶ Run NowGenerates your first digest on the spot. After that it runs every morning on its own.
